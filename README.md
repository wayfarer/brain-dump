# brain-dump

Fresh Next.js project scaffolded with TypeScript, the App Router, Tailwind CSS,
UI primitives, ESLint, Prettier, and Vitest.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run format`
- `npm run format:check`
- `npm run test`
- `npm run test:watch`
